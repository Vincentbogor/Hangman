# Hangman game
A python recreation of the popular game "Hangman" built using turtle graphics. Guess the correct word in 7 attempts before the stickman is fully drawn.

## Versions
- **v1** - Original version, my very first solo project, established the basics, but was very inefficient.
- **v2** - Full refactor. Removed 70% of the code, added themes, difficulty settings, and input validation, resulting in more features for roughly the same amount of code lines.

## How to run
1. Make sure Python is installed
2. Open `hangman_v1.py` (or `hangman_v1.py`) in VS Code
3. Run the file

## What I learned
Refactoring v1 taught me how to replace repetitive if-else logic with loops and lists. This allows for more functions and features to be added for roughly the same amount of code lines leading to a smaller file and added efficiency.
